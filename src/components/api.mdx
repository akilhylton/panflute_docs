## LED Init Function
---

```
void ledInit(void)
```

### Description
* Used to initialize the register for the on-board led.

### No Parameters


## LED Blink Function
---

```
void idle_blink(unsigned int period)
```

### Description 
* Sets how fast the LED blinks. 
### Parameters 
* **period** - The integer that sets the blink speed.
